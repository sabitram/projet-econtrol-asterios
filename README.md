# Projet eControl – Déploiement d’ASTERIOS sur ZUBOARD 1CG

Ce dépôt contient le matériel, les notes et le suivi du projet eControl 2025-2026
réalisé à Centrale Nantes.  
L’objectif principal est le déploiement du système d’exploitation temps réel
**ASTERIOS®** sur une carte **ZUBOARD 1CG** équipée d’un processeur Zynq
(2 cœurs Cortex-A53, 2 cœurs R5 et une zone FPGA).

## Contenu du dépôt

- `documentation/`  
  Documents utilisés et notes diverses.

- `suivi_hebdomadaire/`  
  Comptes rendus hebdomadaires de l’avancement du projet.
